How to Use the Keyboard!

The keyboard can definitely seem daunting with the amount of keys it has, but let's break it down to the basics


These are the letter and number keys, which is how you'll mostly be using your keyboard. Pressing these keys inputs the corresponding letter or number onto your computer


This is the spacebar. Pressing this key will insert a space between your letters.


This is the backspace key. Pressing this will delete the character to the left of the cursor.


This is the enter key. It will create a new line or submits your typing to the webpage


This is the shift key. Holding the shift key while pressing a letter key at the same time will create a capitol letter


These are symbol keys, which contain other characters like punctuation. Pressing the shift key at the same time as a symbol key will print the character at the top of the key
